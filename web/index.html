GSnova is an open source, client–server model web proxy application build on PaaS platforms.

Refer snova site for more information.

You can press the button below to stop gsnova.

Features

Supported Platforms
  • Google Appengine(Java/Go)
  • Java PaaS Platforms(CloudFoundry/Heroku/Openshift/DotCloud/Jelastic)
  • SSH
Enhanced Proxy Auto Config
  • Special PAC config script(json)
  • GFWList buit-in support
  • IP Address Geolocation(Only country)
  • Redirect 3rd proxy(http/socks5)
Anti-interference
  • DNS lookup over trusted DNS servers
  • Remote HTTP DNS lookup support
  • Inject CRLF for special sites
  • Smarthost service support
